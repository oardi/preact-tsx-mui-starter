# Preact TSX MUI Material Starter

Preact TSX, MUI, Webpack5 Starter with SCSS Support

## Usage
* `npm install`
* `npm start`

## Build
* `npm run build`
	* will create a development bundle
* `npm run build:prod`
	* will create a production bundle

## Included
- preact X
- @mui/material
- webpack 5
